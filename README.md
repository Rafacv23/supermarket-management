PWA made with next.js to manage stock and orders of a real supermarket. Easy to use, mobile first and user friendly. 

# Stack
- Next.js
- Turso
- Prisma
- TypeScript
- Zod
- next-pwa
- Jose
- Zustand
- Tailwind
- Shadcn
